Search Help

By default the search will match whole words. Use * to match part of a word (this is called a 'wildcard').

Examples of partial word matches. Respectively, these will match: sparkplug, windshield, tires

spark*
*shield
tire*

You can use what is commonly called "search engine arithmatic" A plus sign + at the beginning of a word will force that word to be included. Only results that contain that word will appear. Likewise a minus - at the beginning of a word will only return results that do not contain that work.

The following search returns information about "fuel" that also includes the word "line":

fuel +line

The following search returns information about "wire" that does not include "spark":

+wire -spark

The minus - is confusing since all non-matching information is returned. It is easy to get back ALL the information in the FAQ database. Use the + as in the example above to require certain terms to get a more restricted search.

You can also use the double quote " to force the search engine to use a phrase instead of matching individual words:

"timing mark"

You can also search only the question or only the answer sections of the database. Use the "question:word" or "answer:word" syntax.

question:running
answer:alternator
